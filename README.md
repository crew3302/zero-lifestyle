# 🛍️ Zero Lifestyle – Modern E-commerce Web Application

Zero Lifestyle is a modern and responsive e-commerce platform built using **React**, **Node.js**, and **MySQL/MongoDB**. It showcases AI-enabled products like smartwatches, headphones, and more — made with performance, design, and user-experience in mind.
